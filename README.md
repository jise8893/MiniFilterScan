# MinifilterScan<br>
MinifilterDriver Scanner IOCP 비동기식 유저모드 어플리케이션과 미니필터 드라이버<br><br>
사용자의 파일 확장자를 검사한 후 해당 파일의 내용을 읽어 SHA256 으로 해싱 후<br>유저모드 어플리케이션에 버퍼값을 전송해 어플리케이션 실행을 차단하는 프로젝트
